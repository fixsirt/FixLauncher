# FixLauncher v2.0.8 — Что нового

## 🆕 Добавлено
- **Инстансы** — несколько профилей игры с разными версиями, создание, копирование, экспорт/импорт ZIP
- **Скриншоты** — просмотр всех скриншотов из всех версий прямо в лаунчере
- **Текстуры и шейдеры** — установка ресурспаков и шейдерпаков через Modrinth
- **Все загрузчики** — Fabric, Forge, NeoForge, Quilt в одном списке версий
- **Активность за неделю** — блок с игровым временем по дням на главной
- **Запуск без консоли** — `launch.bat` открывает лаунчер без чёрного окна

## ⚡ Улучшено
- Все вкладки загружаются при старте — переключение мгновенное, без задержек
- Анимации стали плавнее, убраны рывки при переключении вкладок
- Смена темы теперь с ripple-эффектом от кружка

## 🗑️ Удалено
- VanillaSuns / Evacuation — убрана полностью

## 🐛 Исправлено
- Кнопка «Инфо» у мода не работала
- Инстанс показывал «Vanilla» вместо реального загрузчика
- Название инстанса отображалось транслитом вместо русского
- Скриншоты мигали при повторном клике на вкладку
